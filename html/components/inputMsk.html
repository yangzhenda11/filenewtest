固定格式文本 API

基于inputmask.js

Options:
name type default 描述
mask string '' 格式化字符串，可比对的字符见Format表
placeholder string '_' 输入框中显示的字符 
$('.inputmask').inputmask({
  mask: '999-99-999-9999-9'
})
Format:匹对类型
Character Description
9 Number
a Letter
w Alphanumeric
* 任何字符
? 0~1个可选字符
method:调用方法
$(selector).inputmask({ mask: "99-999-99"}); 创建inputmask
$(selector).inputmask('unmaskedvalue'); 获取不带标记的值，即用户输入的值
$(selector).inputmask('setvalue',value); 赋值
$(selector).inputmask('remove'); 去除inputmask
$("#test").inputmask("getemptymask") 获取空标记
$("#test").inputmask("hasMaskedValue") 是否有符合校验的值
$(selector).inputmask("isComplete") 验证当前值是否已完成
数字格式
mask "9" or mask "99" or ... mask "9999999999"
123456 => ￥ ___,__1,234.56
只能数字，且小数点后最多两位
